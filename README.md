## 🚀 HOW TO USE

### For Kids (Students):

1. **Double-click** `LAUNCH_ACADEMY.bat` (recommended, no Python required) OR
2. **Open** `index.html` in any web browser

3. **Select your avatar** (Zander, Zeke, Obi, or Jedi)

4. **Complete missions in order:**
   - Click on a mission card
   - Watch/read the lesson content
   - Complete all activities (check them off)
   - Spend at least 60 minutes (timer tracks this!)
   - Take the knowledge quiz (need 80% to pass)
   - Write in your reflection journal
   - Click "Complete Mission"

5. **Week 9 Final Exam:**
   - Complete the pre-flight checklist
   - All 8 items must be checked
   - Launch your rocket!
   - Earn your certificate!

### For Parents/Teachers (Coach Cisco Dashboard):

1. Click "Coach Dashboard" in the navigation
2. View student progress:
   - Weeks completed
   - Total time spent
   - Average quiz scores
   - Detailed history table
3. Reset data if needed (for new students)

## 📞 SUPPORT & CUSTOMIZATION

**To modify content:**
- Open `RocketryAcademy.html` in a text editor
- Search for `const MISSIONS` to edit lessons
- Search for `const QUIZ_DATA` to edit quizzes
- Save and refresh browser

**To replace avatars:**
- Replace PNG files in `assets/avatars/` (keep same filenames)
- Or update the paths in `const STUDENTS` and `const COACH` in `RocketryAcademy.html`

**To translate:**
- All text is in the JavaScript section
- Look for text strings and translate as needed
