================================================================================
                         DEPLOYMENT INSTRUCTIONS
================================================================================

FOR SCHOOL/COMMUNITY CENTER USE:

1. COPY THE FOLDER
   - Copy entire "RocketryAcademy" folder to computer
   - Can be on desktop, documents, or USB drive
   - All files must stay together

2. LAUNCH THE APPLICATION
   - Double-click "LAUNCH_ACADEMY.bat" (Windows)
   - OR open "index.html" in browser

3. FIRST-TIME SETUP
   - Ensure the `assets/` folder is present beside RocketryAcademy.html
   - Select student avatar
   - Click "Begin Training"

4. WEEKLY USAGE
   - Student selects their avatar
   - Complete one mission per week
   - Spend 60-90 minutes on activities
   - Pass quiz (80% minimum)
   - Write reflection
   - Mark mission complete

5. PARENT/TEACHER MONITORING
   - Click "Coach Dashboard" in navigation
   - Review time spent, scores, progress
   - Reset data for new students

6. FINAL CERTIFICATION
   - Complete all 9 weeks
   - Pass Week 9 pre-flight checklist
   - Certificate generates automatically
   - Print or save as PDF

FOR HOME USE:
Same as above, but simpler:
- Launch → Select Avatar → Complete Missions → Earn Certificate

================================================================================
                         TROUBLESHOOTING GUIDE
================================================================================

PROBLEM: Avatars don't show
SOLUTION: Verify `assets/avatars/` exists and contains all PNG files
          Reload page (Ctrl+F5 or Cmd+Shift+R)

PROBLEM: Logo doesn't load
SOLUTION: Verify `assets/warriors.png` exists beside RocketryAcademy.html

PROBLEM: Progress doesn't save
SOLUTION: Check that browser JavaScript is enabled
          Check that localStorage is enabled
          Try different browser

PROBLEM: Quiz won't submit
SOLUTION: Make sure all 5 questions are answered
          Check browser console for errors (F12)

PROBLEM: Timer doesn't start
SOLUTION: Make sure you're on a mission detail page
          Check that JavaScript is enabled

PROBLEM: Certificate won't display
SOLUTION: Must complete Week 9 pre-flight checklist
          All 8 items must be checked

PROBLEM: Can't access Coach Dashboard
SOLUTION: Click "Coach Dashboard" button in navigation
          It's available from any screen
